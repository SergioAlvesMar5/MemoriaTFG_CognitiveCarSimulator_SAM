BP_TrafficLight:

StaticMesh:

StopLinePosition arrow justo debajo del semáforo

SensorZone boxcollision que cubre toda la zona de parada del semaforo



variables globales:

DMI_Roja MaterialIntanceDynamic

DMI_Verde MaterialIntanceDynamic

DMI_Amarilla MaterialIntanceDynamic 
 
bIsRedLight boolean true 

CurrentState E_TrafficLightState

ColorState float 0.0

ViolationPenaltyMultiplier float 3.0


-----------------------------------------------------------------------------------------

EventGraph:



- OnComponentBeginOverlap(SensorZone): llama a UpdateTrafficZone



- OnComponentEndOverlap(SensorZone): casttobpvehicleai del otheractor. Del getaicontroller del cast se hace un casttobpaicontroller. luego hay un Branch, cuya condición es (TrafficLigthState del aicontroller == 0.0)AND(GetForwardspeeddelotheractor>5.0). Si el Branch es false, setea AproachSpeed del MycontrolledCar del aicontroller como 0.0, luego del cast se setea IsTrafficLightActive del aicontroller en false y TrafficLightState del aicontroller como 1.0. Si es True, setea Fitness del OtherActor como Fitness - (((Clamp(value=(forwardspeed/maxspeed),min=0.0,max=1.0))^2) x maxspeed x ViolationPenaltyMultiplier), hace un Set DebugDeathPenalty del cast de VehicleAi como (((Clamp(value=(forwardspeed/maxspeed),min=0.0,max=1.0))^2) x maxspeed x ViolationPenaltyMultiplier), y llama a la función KillCar del BP_VehicleAI cuya razón de muerte es "TRAFFIC_VIOLATION". 



- UpdateTrafficZone: foreachloop de Getoverlappingactors(target=SensorZone, class=bpvehicleai), cuyo loop body es: casttobpaicontroller con el getaicontroller del arrayelement. Del cast se setea StopTargetLocation como GetWorldposition(StopLinePosition), se setea IsTrafficLightActive como true y se setea TrafficLightState como ColorState, luego hay un Branch que si ColorState==0.0, setea AproachSpeed del MyControlledCar del castaicontroller como GetForwardSpeed del mycontrolledCar.



- SetLightState:

input:

newstate E_TrafficLightState 

flujo: 

setea CurrentState como Newstate. luego hay un Switch on E_TrafficLightState: 

- si es red: primero setscalarparametervalue(target=DMI_Roja, parametervalue=brillo, value=50.0), luego setscalarparametervalue(target=DMI_Amarilla, parametervalue=brillo, value=0.0), luego setscalarparametervalue(target=DMI_Verde, parametervalue=brillo, value=0.0), luego setea ColorState como 0.0 y llama a UpdateTrafficZone.

- si es green: primero setscalarparametervalue(target=DMI_Roja, parametervalue=brillo, value=0.0), luego setscalarparametervalue(target=DMI_Amarilla, parametervalue=brillo, value=0.0), luego setscalarparametervalue(target=DMI_Verde, parametervalue=brillo, value=50.0), luego setea ColorState como 1.0 y llama a UpdateTrafficZone.

- si es amber: primero setscalarparametervalue(target=DMI_Roja, parametervalue=brillo, value=0.0), luego setscalarparametervalue(target=DMI_Amarilla, parametervalue=brillo, value=50.0), luego setscalarparametervalue(target=DMI_Verde, parametervalue=brillo, value=0.0), luego setea ColorState como 0.5 y llama a UpdateTrafficZone.

BP_Sign_Stop:

DefaultSceenRoot:
…
SensorZone boxcollision que cubre toda la zona de parada del stop

StopLine arrow justo al final del stop

CrossingZone boxcollision que cubre la zona de intersección 




variables:

CarsAlivedInCrossingZone integer 0

ViolationPenaltyMultiplier float 2.0




EventGraph:


- OnComponentBeginOverlap(SensorZone): casttoBpVehicleAi de otheractor. del cast se coge aicontroller y se hace un casttobpaicontroller. Del cast se setea StopTargetLocation como GetWorldposition(StopLine), se setea ForceStop como True, se setea StoppedCorrectly del casttoaicontroller como false, setea AproachSpeed del MyControlledCar del castaicontroller como GetForwardSpeed del mycontrolledCar, y se llama a WaitforFullStop con el targetCar como el casttoBpVehicleAi.



- OnComponentEndOverlap(SensorZone): casttobpvehicleai del otheractor. Del getaicontroller del cast se hace un casttobpaicontroller. luego un Branch cuya condición es StoppedCorrectly del aicontroller. Si es True, setea AproachSpeed del MycontrolledCar del aicontroller como 0.0, luego del cast se setea forcestop como false. Si es false, setea Fitness como Fitness - (((Clamp(value=(forwardspeed/maxspeed),min=0.0,max=1.0))^2) x maxspeed x ViolationPenaltyMultiplier), hace un Set DebugDeathPenalty de VehicleAi como (((Clamp(value=(forwardspeed/maxspeed),min=0.0,max=1.0))^2) x maxspeed x ViolationPenaltyMultiplier), y llama a la función KillCar del BP_VehicleAI cuya razón de muerte es "STOP_VIOLATION".



- WaitForFullStop: 

inputs: 

TargetCar BP_VehicleAi

lo primero un delay de 0.1s, luego hay un isvalid? de TargetCar, si es valid, hay un Branch cuya condition es ((ABS(ForwardSpeed)<5.0) AND (dot(GetActorLocation(TargetCar)-StopTargetLocationdeaicontrollerdetargetcar, GetForwardVector(StopLine))<=0.0) AND ((1.0 - ClampFloat(value=ABS(dot(GetActorLocation(TargetCar)-StopTargetLocationdeaicontrollerdetargetcar, GetForwardVector(StopLine)))/MaxVisionDistancedelaicontrollerdeltargetcar, min=0.0, max=1.0)) >= (1-(ClampFloat(value=SafeDivide(a=ABS(ForwardSpeed), b=AproachSpeed), min=0.0, max=1.0))))). 

Si es false llama a WaitForFullStop de nuevo con el mismo TargetCar, si es true, primero hace un delay de 3.0s, luego hay otro isvalid? de TargetCar. Si es valido, hay un Branch cuya condition es ((ABS(ForwardSpeed)<5.0) AND (dot(GetActorLocation(TargetCar)-StopTargetLocationdeaicontrollerdetargetcar, GetForwardVector(StopLine))<=0.0) AND ((1.0 - ClampFloat(value=ABS(dot(GetActorLocation(TargetCar)-StopTargetLocationdeaicontrollerdetargetcar, GetForwardVector(StopLine)))/MaxVisionDistancedelaicontrollerdeltargetcar, min=0.0, max=1.0)) >= (1-(ClampFloat(value=SafeDivide(a=ABS(ForwardSpeed), b=AproachSpeed), min=0.0, max=1.0))))), Si es false llama a WaitForFullStop de nuevo con el mismo TargetCar, si es true, con el aicontorller de Targetcar se setea StoppedCorrectly como true y luego llama a CheckCrossingClear. 



- CheckCrossingClear:

inputs: 

TargetCar BP_VehicleAi

lo primero, setea CarsAlivedInCrossingZone como 0. Luego hay un foreachloop de GetOverlappingActorsCrossingZone(BP_VehicleAI), cuyo loop body es: un casttoBPvehicleAI, del cast sacamos IsDead, que sirve de condition para un Branch. Si es true vacío, si es false hace un ++ de CarsAlivedInCrossingZone. Cuando se completa el loop, hay un Branch cuya condition es CarsAlivedInCrossingZone>0, si es true, hace un delay de 0.5s y vuelve a llamar con el mismo TargetCar a CheckCrossingClear. Si es false, primero hay un isvalid? de target car, y luego si es valido, hace un casttobpaicontroller con el getaicontroller del targetcar. Luego, setea AproachSpeed del MycontrolledCar del aicontroller como 0.0 y se setea forcestop del cast como false.
