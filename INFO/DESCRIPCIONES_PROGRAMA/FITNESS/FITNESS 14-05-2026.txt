EventGraph:





BeginPlay:

lo primero hay un Parent:BeginPlay, luego hay un CasttoBP_aicontroller de GetController. Del cast lo primero hay un isvalid?, si es así se setea AiController con el cast. Luego hay un setColor. Luego hay un Casttochaoswheeledvehiclemovementcomponent de vehiclemovementcomponent. Del cast se setea AsChaosWheeledVehicleMovementComponent. 

Luego se setea LastLocation con GetActorLocation. Después de eso hay un SetTimerbyEvent, que tiene como opciones Time 1.0, y looping en True. Por último se setea CheckStuckTimerHandle como el returnvalue del SetTimerByEvent.




EventTick:

Lo primero hay un Parent:Tick de 0.0. Luego hay un SetAngularDamping cuyo target=mesh, InDamping=(SelectFloat(a=0.0, b=3.0, picka=Ismovingonground(AsChaosWheeledVehicleMovementComponent))). Luego hay un Branch cuya condition es NOT(IsDead), si es false, vacío, si es true sigue el flujo. Luego hay un Branch cuya condition es (NOT(TrainingMode de AiController)) AND (TimeAlived>MaxTimeAlived). Si es true llama a KillCar cuya razón de muerte es TIMEFINISHED, si es false, pasa a otro branch cuya condition es TimeAlived>120. Si es true, llama a KillCar cuya razón de muerte es TIMEFINISHED, si es false, llama a la función UpdateFitness.




CheckStuck:

Está conectada con el hilo rojo que sale del SetTimerbyEvent. 

El flujo de este evento es: Branch cuya condition es IsDead, si es true hace un ClearandInvalidateTimerbyHandle(CheckStuckTimerHandle), si es false, time alived ++. Luego un Branch cuya condición es si time alived es mayor que 3. Si el Branch es false, setea WasInStopContextLastCheck como (ForceStopdelAiController) OR ((bIsTrafficLightActivedelAiController) AND ((TrafficLightStatedelAiController)<=0.5)), setea TimeSinceLastMove como 0 y setea LastLocation como getactorlocation. 

Si es true, primero pasa a un branch cuya condition es NOT(NOT(WasInStopContextLastCheck) AND (ForceStop OR (IsTrafficLightActive AND (TrafficLightState<=0.5)))), si es true pasa al branch A, si es false, hace un ++ de Debug_NumStopZonesEntered y luego pasa al Branch A.

Luego, pasa a un Branch (llamémosle A) cuya condition es NOT(NOT((ForceStop del AiController) OR ((bIsTrafficLightActivedelAiController) AND ((TrafficLightStatedelAiController)<=0.5))) AND WasInStopContextLastCheck). 

Si A es false, setea ResumeGraceActive como True, setea ResumeGraceTime como ClampFloat(value=SafeDivide(a=MAX(SensorEntryDistancedelAiController, 1.0), b=MAX(AproachSpeed, 1.0)), min=1.0, max=3.0), luego setea TimeSinceLastMove, WasInStopContextLastCheck y setea LastLocation.

Si A es true, pasa a un Branch (llamémosle B) cuya condition es NOT(ResumeGraceActive). Si B es False, setea ResumeGraceTime como ResumeGraceTime - 1.0. Luego, hay Branch cuya condition es Throttledelaicontroller > 0.0, si es True, setea Debug_Acum_ThrottleDuringGraceTime como Debug_Acum_ThrottleDuringGraceTime + Throttledelaicontroller, si es false, setea Debug_Acum_BrakeDuringGraceTime como Debug_Acum_BrakeDuringGraceTime + ABS(Throttledelaicontroller). Luego pasa a un Branch (llamémosle C), cuya condition es ResumeGraceTime <= 0.0. Si es false, sigue el flujo, si es true, setea ResumeGraceActive como False. Tras ambos caminos, setea TimeSinceLastMove, WasInStopContextLastCheck y setea LastLocation.

Si B es true, pasa a un Branch (llamemosle D), cuya condición es: ((DistanceVector(GetActorLocation, LastLocation)) < (selectfloat (a=(maxspeedx0.05), b=(maxspeedx0.05xfrontobstacledistance), picka=(frontobstacledistance>0.65))) AND (NOT(ForceStop del AiController)) AND (NOT((bIsTrafficLightActive del AiController) AND ((TrafficLightState del AiController)<=0.5))). 

Si el branch D es True, hace un ++ de TimeSinceLastMove, y luego setea DeltaLazyPenalty como MAX(0.0, (selectfloat (a=(maxspeedx0.05), b=(maxspeedx0.05xfrontobstacledistance), picka=(frontobstacledistance>0.65)) - DistanceVector(GetActorLocation, LastLocation))), luego setea Fitness como Fitness - DeltaLazyPenalty, setea Debug_Acum_LazyPenalty como Debug_Acum_LazyPenalty + DeltaLazyPenalty. 

Por último hay un Branch (E) cuya condition es TimeSinceLastMove>=3, si es false setea WasInStopContextLastCheck y setea LastLocation, si es true, setea Debug_DeathPenalty como DeltaLazyPenalty y llama a killcar cuya cause es "LAZY". 

Si el branch D es false, setea TimeSinceLastMove, WasInStopContextLastCheck y setea Last Location.




EventActorBeginOverlap:

Lo primero hay un isvalid? de otheractor. Luego hay un CasttoBPIntersectionBorders con el otheractor. Si el cast es válido, pasa a un branch (llamemosle A), cuya condition es Trigger del cast == CurrentTrigger del AiController. Si A es true pasa a otro branch (llamemosle B) cuya condition es DirectionIntent del cast == IntersectionDecision de AiController. Si B es True, se setea DeltaCollisionDeathPenalty como ((getforwardspeed^2/(maxspeed del aicontroller))x(1-(frontobstacledistnace del aicontroller))). Luego se setea Fitness como fitness - DeltaCollisionDeathPenalty. Luego se setea DebugDeathPenalty como DeltaCollisionDeathPenalty. Luego se llama a KillCar cuya Cause es "COLLISION_WITH_" + getDisplayName de otherActor.

Si A o B son false, pasan a un branch (llamemosle C) cuya condition es ActorHasTag=Semaforo del otheractor. Si es true, hace un casttoBPTrafficLight y del cast setea ActualTrafficLight.

Si el cast de intersectionBorders es false, pasa a un branch cuya condition es (ActorHasTag=Obstaculo) OR (ActorHasTag=Discontinua). Si es true pasa a setear el fitness y matar al coche, si es false, pasan al branch C.




EventActorEndOverlap:

Lo primero hay un CastToTrafficLight, luego hay un branch cuya condition es si el cast == ActualTrafficLight. Si false, vacío, si es true SeteaActualTrafficLight como vacío y setea ForceStop del AiController como False.




KillCar:

input:

Cause String 


Lo primero hay un ++ de TimeDeath. Luego hay un Branch cuya condition es TimeDeath > 1. Si es true vacío, si es false, hay un -- de Time alived, y setea IsDead como true. Luego setea DeathReason con Cause. Luego hay un SetSimulatePhysics cuyo target es Mesh y lo setea en False. Luego hace un SetColorRed. Por último hace un SaveStringTexttoFile, con filename=Fitness_Debug.csv, savetext que guarda GenerationCount, GetDisplayName(self), Fitness, TimeAlived, SpawnPoint, DeathReason, DebugDeathPenalty, DebugAcumPenaltyVelocidad, DebugAcumA, (Debug_Acum_B_Sum/MAX(Debug_TicksNormal,1.0)), DebugAcumC, DebugAcumD, DebugAcumE, DebugAcumF, DebugAcumWaitBonus, DebugAcumNetNormal, DebugAcumPenaltyCorrectingWrong, DebugAcumPenaltyTrafficViolation, DebugTicksNormal, DebugTicksTotal, TrainingModedelAicontroller, Debug_Acum_ThrottlePositive, Debug_Acum_BrakeInput, Debug_Acum_CoastTime, Debug_Acum_StopContextBrake, Debug_Acum_StopContextThrottle ,Debug_Ticks_StopContext, Debug_Acum_CreepingPenalty, Debug_Acum_ReversePenalty, Debug_Acum_LazyPenalty, Debug_Acum_SteeringWhileStopped, Debug_Acum_ThrottleDuringGraceTime, Debug_Acum_BrakeDuringGraceTime, YieldValidationTime, StopValidationTime, Debug_Acum_SteeringInput, Debug_Acum_TargetSteering, Debug_CarIndex, Debug_Acum_ABS_Steering, Debug_Ticks_StopContext_TrafficLight, Debug_Ticks_StopContext_Stop, Debug_Ticks_StopContext_Yield, Debug_MutNumChanged, Debug_MutAvgMagnitude, Debug_MutNumLarge, Debug_MutParentFitness, Debug_Acum_SteerApproachPenalty, Debug_NumStopZonesEntered, Debug_NotNorm_PeakFitness. flujo:

Lo primero hay un Branch cuya condition es getforwardspeed >= (maxspeed(1000)*frontobstacledistance(1 lejos, 0 cerca)), si es false sigue el flujo, si es true se setea DeltaPenaltyVelocidad como (MAX(0.0, (ClampFloat(value=(forwardspeed/maxspeed), min=0.0, max=1.0) - FrontObstacleDistance)))^2 * worlddeltaseconds * maxspeed, luego se setea el fitness como fitness - DeltaPenaltyVelocidad, y luego se setea DebugAcumPenaltyVelocidad como DebugAcumPenaltyVelocidad + DeltaPenaltyVelocidad, y sigue el flujo.


Luego, hay un Branch cuya condition es forwardspeed < 0.0, si es false, setea ReverseTime como 0.0 y sigue el flujo. Si es true, setea ReverseTime como ReverseTime + WorldDeltaSeconds, luego setea DeltaReversePenalty como ((Clamp(value=(ABS(forwardspeed)/maxspeed),min=0.0,max=1.0))^2)*worlddeltaseconds*maxspeed), luego setea Fitness como Fitness - DeltaReversePenalty, y setea Debug_Acum_ReversePenalty como Debug_Acum_ReversePenalty + DeltaReversePenalty.


tras ambos caminos, luego está el Branch cuya condición es si (frontobstacledistance>=0.99 AND steering*horizontaldistdiff(que es distsensordcha-distsensorizda)<0.0 AND ABS(horizontaldistdiff) > 0.075) OR ((forwardspeed < -5.0) AND (SafeDivide(A=(ABS(ForwardSpeed)*ReverseTime), B=MAX(FrontObstacleDistance*MaxVisionDistance, 1.0)) > Clamp(SafeDivide(A=ABS(ForwardSpeed), B=MAX(AproachSpeed, 1.0)), 0.0, 1.0))). 

si es true, hay otro Branch cuya condición es si (frontobstacledistance>=0.99 AND steering*horizontaldistdiff(que es distsensordcha-distsensorizda)<0.0 AND ABS(horizontaldistdiff) > 0.075) AND ((forwardspeed < -5.0) AND (SafeDivide(A=(ABS(ForwardSpeed)*ReverseTime), B=MAX(FrontObstacleDistance*MaxVisionDistance, 1.0)) > Clamp(SafeDivide(A=ABS(ForwardSpeed), B=MAX(AproachSpeed, 1.0)), 0.0, 1.0))). 

si es true, setea DeltaDeathPenalty como (ABS(forwardspeed)+(maxvisiondistance(3000.0)*ABS(steering)*ABS(horizontaldistancediff))), luego seteea fitness como fitness - DeltaDeathPenalty, setea DebugDeathPenalty como DeltaDeathPenalty y llama a KillCar, esta razón de muerte es REVERSING_WRONG.

Si el Branch es false, pasa a otro Branch cuya condición es (forwardspeed < -5.0) AND (SafeDivide(A=(ABS(ForwardSpeed)*ReverseTime), B=MAX(FrontObstacleDistance*MaxVisionDistance, 1.0)) > Clamp(SafeDivide(A=ABS(ForwardSpeed), B=MAX(AproachSpeed, 1.0)), 0.0, 1.0)). Si es true, setea DeltaDeathPenalty como ABS(forwardspeed), fitness como fitness - DeltaDeathPenalty, setea DebugDeathPenalty como DeltaDeathPenalty y llama a killcar, esta razón de muerte es REVERSE. 

Si el Branch es false, setea DeltaPenaltyCorrectingWrong como (maxvisiondistance(3000.0)*ABS(steering)*ABS(horizontaldistancediff)*worlddeltaseconds), luego setea fitness como fitness - DeltaPenaltyCorrectingWrong. Luego setea DebugAcumPenaltyCorrectingWrong como DebugAcumPenaltyCorrectingWrong + DeltaPenaltyCorrectingWrong y pasa al flujo de ++ de DebugTicksTotal, Debug_Acum_SteeringInput... 





En caso de que el Branch de (frontobstacledistance>=0.99 AND steering*horizontaldistdiff(que es distsensordcha-distsensorizda)<0.0 AND ABS(horizontaldistdiff) > 0.075) OR ((forwardspeed < -5.0) AND (SafeDivide(A=(ABS(ForwardSpeed)*ReverseTime), B=MAX(FrontObstacleDistance*MaxVisionDistance, 1.0)) > Clamp(SafeDivide(A=ABS(ForwardSpeed), B=MAX(AproachSpeed, 1.0)), 0.0, 1.0))) salga false, se pasa a otro Branch, dentro ya del cálculo normal del fitness:


hay un Branch (A) cuya condition es ForceStopdeaicontroller OR (IsTrafficLightActivedelaicontroller AND (TrafficLightStatedelaicontroller<=0.5)), si es false sigue el flujo (el Branch de waitbonus), si es true, pasa a otro Branch (B).

Branch (B) cuya condition es ((bIsTrafficLightActive del aicontroller) AND ((TrafficLightState del aicontroller)<=0.5), si es false sigue el flujo, si es true, setea DeltaPenaltyTrafficViolation como (((Clamp(value=(forwardspeed/maxspeed),min=0.0,max=1.0))^2)*worlddeltaseconds*maxspeed*SelectFloat(a=0.3, b=1.0, picka=(TrafficLightState==0.5), luego setea Fitness como Fitness - DeltaPenaltyTrafficViolation y setea Debug_Acum_PenaltyTrafficViolation como Debug_Acum_PenaltyTrafficViolation + DeltaPenaltyTrafficViolation.

tras ambos caminos del branch B, hay un branch (C), cuya condition es (steering * horizontaldistdiff) < 0.0. Si es false pasa al Branch D, si es true, setea DeltaSteeringWhileStopped como (ABS(Steeringdeaicontroller) * Deltaseconds * MaxSpeeddelaicontroller), luego setea Fitness como Fitness - DeltaSteeringWhileStopped, y setea Debug_Acum_SteeringWhileStopped como Debug_Acum_SteeringWhileStopped + DeltaSteeringWhileStopped. 

tras ambos caminos del branch C, se setea DeltaSteerApproachPenalty como (ClampFloat(value=(ForwardSpeed/MaxSpeed), min=0.0, max=1.0) x MaxSpeed x ClampFloat(value=(AproachSpeed/MaxSpeed), min=0.0, max=1.0) x ABS(Steering) x WorldDeltaSeconds), luego se setea Fitness como Fitness - DeltaSteerApproachPenalty, luego se setea Debug_Acum_SteerApproachPenalty como Debug_Acum_SteerApproachPenalty + DeltaSteerApproachPenalty. Luego, hay un branch (D), cuya condition es TrafficControlTypedelaicontroller<=1 AND (ForceStop OR (IsTrafficLightActive AND TrafficLightState <= 0.5)). si es false sigue el flujo (el Branch de waitbonus).

Si es true, se setea DeltaCreepingPenalty como Clamp(value=SafeDivide(a=ABS(GetForwardSpeed), b=MAX(AproachSpeed, 1.0)), min=0.0, max=1.0) x MAX(AproachSpeed, 0.0) x DeltaSeconds x (1- ClampFloat(value=SafeDivide(a= Distance(GetActorLocation, StopTargetLocationdelAiController), b=MAX(Sensorentrydistancedeaicontroller, 1.0)), min=0.0, max=1.0)) x . Luego hay un set Fitness como Fitness - DeltaCreepingPenalty. tras eso se setea Debug_Acum_CreepingPenalty como Debug_Acum_CreepingPenalty + DeltaCreepingPenalty.

Luego hay otro Branch cuya condition es (ForceStopdeaicontroller OR (IsTrafficLightActivedelaicontroller AND TrafficLightStatedelaicontroller<=0.5)) AND (ABS(getfordwardspeed)<(Maxspeeddelaicontroller*0.005)), si es false sigue el flujo. 
si es true setea DeltaWaitBonus como (((LERP(A=1.0, B=(1-(ABS(distsensordcha-distsensorizda))), alpha=frontobstacledistance) x Getworlddeltaseconds x AproachSpeed x frontobstacledistance)+(((Clamp(value=(aproachspeed/maxspeed),min=0.0,max=1.0)^2)*maxspeed*frontobsacledistance^2*worlddeltaseconds))) x (1-ClampFloat(value=SafeDivide(Distance(GetActorLocation, StopTargetLocationdelaicontroller), MAX(SensorEntryDistancedelacontroller, 1.0)), min=0.0, max=1.0)). Luego setea fitness como Fitness + DeltaWaitBonus. Luego setea Debug_Acum_WaitBonus como Debug_Acum_WaitBonus + DeltaWaitBonus. Ahora, se pasa al calculo del fitness:



primero se setea DeltaA como getforwardspeed x LERP(A=1.0, B=(1-(ABS(distsensordcha-distsensorizda))), alpha=frontobstacledistance) x frontobstacledistance x Getworlddeltaseconds.

luego se setea DeltaB como MAX(0.0,(PrevHorizontalDistanceDiference - ABS(HorizontalDistDiff))).

luego se setea DeltaC como (1.0/MAX(RoadWidth,0.05)).

luego se setea DeltaD como (1.0 / (maxsteeringangle->D2R->SIN))).

luego se setea DeltaE como DeltaB x DeltaC x DeltaD.

luego se setea DeltaF como (1 - (ClampFloat(value=ABS(ClampFloat(value=(forwardspeed/maxspeed), min=0.0, max=1.0) - FrontObstacleDistance), min=0.0, max=1.0))) * ClampFloat(value=(forwardspeed/maxspeed), min=0.0, max=1.0) * maxspeed * frontobsacledistance^2 * worlddeltaseconds * MAX(Clamp(value=(ForwardSpeed/Maxspeeddelaicontroller), min=0.0, max=1.0), Clamp(value=Throttledelaicontroller, min=0.0, max=1.0)).

luego se setea DeltaNetNormal como DeltaA x DeltaE x DeltaF.

luego se setea Fitness como Fitness + DeltaNetNormal. 

Después se setea Debug_Acum_A como Debug_Acum_A + DeltaA, se setea Debug_Acum_B_Sum como Debug_Acum_B_Sum + DeltaB, setea Debug_Acum_C como Debug_Acum_C + DeltaC, se setea Debug_Acum_D como Debug_Acum_D + DeltaD, se setea Debug_Acum_E como Debug_Acum_E + DeltaE, se setea Debug_Acum_F como Debug_Acum_F + DeltaF, se setea Debug_Acum_NetNormal como Debug_Acum_NetNormal + DeltaNetNormal.

Después hay un ++ DebugTicksNormal. Luego, hay un ++ DebugTicksTotal. 

Luego se setea Debug_Acum_SteeringInput como Debug_Acum_SteeringInput + Steeringdelaicontroller. Luego, se setea Debug_Acum_ABS_Steering como Debug_Acum_ABS_Steering + ABS(Steeringdelaicontroller). Luego, se setea Debug_Acum_TargetSteering como Debug_Acum_TargetSteering + TargetSteeringdelAicontroller. Luego, hay un Branch cuya condition es Throttle>0.0, si es true setea Debug_Acum_ThrottlePositive como Debug_Acum_ThrottlePositive + Throttle*DeltaSeconds. Si es false, setea Debug_Acum_BrakeInput como Debug_Acum_BrakeInput + ABS(Throttle)*DeltaSeconds. Tras ambos caminos, hay un Branch cuya condition es ABS(Throttle)<=0.05. Si es false sigue el flujo, si es true setea Debug_Acum_CoastTime como Debug_Acum_CoastTime + DeltaSeconds.

Tras eso, hay un Branch cuya condition es (ForceStopdeaicontroller OR (IsTrafficLightActivedelaicontroller AND TrafficLightStatedelaicontroller<=0.5)), si es false sigue el flujo. Si es true, pasa a otro Branch cuya condition es TrafficControlTypedelaicontroller == 0, si es true hace un ++ de Debug_Ticks_StopContext_TrafficLight, si es false pasa a otro Branch cuya condition es TrafficControlTypedelaicontroller == 1, si es true hace un ++ de Debug_Ticks_StopContext_Stop, si es false, hace un ++ de Debug_Ticks_StopContext_Yield.

Después de setear el tipo de StopContext, hace un ++ de Debug_Ticks_StopContext y pasa a otro Branch, cuya condition es Throttle>0.0. Si es true, setea Debug_Acum_StopContextThrottle como Debug_Acum_StopContextThrottle + Throttle*DeltaSeconds. Si es false, setea Debug_Acum_StopContextBrake como Debug_Acum_StopContextBrake + ABS(Throttle)*DeltaSeconds.


PrevHorizontaldistancediference se setea al final, cuyo valor es ABS(dist sensor dcha - dist sensor izda) y por ultimo, se setea Debug_NotNorm_PeakFitness como MAX(Debug_NotNorm_PeakFitness Fitness).
