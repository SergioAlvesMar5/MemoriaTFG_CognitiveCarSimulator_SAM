EventGraph:





BeginPlay:

lo primero hay un Parent:BeginPlay, luego hay un CasttoBP_aicontroller de GetController. Del cast lo primero hay un isvalid?, si es así se setea AiController con el cast. Luego hay un setColor. Luego hay un Casttochaoswheeledvehiclemovementcomponent de vehiclemovementcomponent. Del cast se setea AsChaosWheeledVehicleMovementComponent. Luego, se setea NavPenaltyDeathThreshold como MaxVisionDistancedelAiController^2 / MaxSpeeddelAiController. 

Luego se setea LastLocation con GetActorLocation. Después de eso hay un SetTimerbyEvent, que tiene como opciones Time 1.0, y looping en True. Por último se setea CheckStuckTimerHandle como el returnvalue del SetTimerByEvent.




EventTick:

Lo primero hay un Parent:Tick de 0.0. Luego hay un SetAngularDamping cuyo target=mesh, InDamping=(SelectFloat(a=0.0, b=3.0, picka=Ismovingonground(AsChaosWheeledVehicleMovementComponent))). Luego hay un Branch cuya condition es NOT(IsDead), si es false, vacío, si es true sigue el flujo. Luego hay un Branch cuya condition es (NOT(TrainingMode de AiController)) AND (TimeAlived>MaxTimeAlived). Si es true llama a KillCar cuya razón de muerte es TIMEFINISHED, si es false, pasa a otro branch cuya condition es TimeAlived>600. Si es true, llama a KillCar cuya razón de muerte es TIMEFINISHED, si es false, llama a la función UpdateFitness.




CheckStuck:

Está conectada con el hilo rojo que sale del SetTimerbyEvent. 

El flujo de este evento es: Branch cuya condition es IsDead, si es true hace un ClearandInvalidateTimerbyHandle(CheckStuckTimerHandle), si es false, time alived ++. Luego un Branch cuya condición es si time alived es mayor que 3. Si el Branch es false, setea TimeSinceLastMove como 0, setea WasInStopContextLastCheck como (ForceStopdelAiController) OR ((bIsTrafficLightActivedelAiController) AND ((TrafficLightStatedelAiController)<=0.5)), setea LastLocation como getactorlocation y llama a CheckNavViolation(SafeDelta = 1.0, bCountSeconds = true). 

Si es true, primero pasa a un branch cuya condition es NOT(NOT(WasInStopContextLastCheck) AND (ForceStop OR (IsTrafficLightActive AND (TrafficLightState<=0.5)))), si es true pasa al branch A, si es false, hace un ++ de Debug_NumStopZonesEntered y luego pasa al Branch A.

Luego, pasa a un Branch (llamémosle A) cuya condition es NOT(NOT((ForceStop del AiController) OR ((bIsTrafficLightActivedelAiController) AND ((TrafficLightStatedelAiController)<=0.5))) AND WasInStopContextLastCheck). 

Si A es false, setea ResumeGraceActive como True, setea ResumeGraceTime como ClampFloat(value=SafeDivide(a=MAX(SensorEntryDistancedelAiController, 1.0), b=MAX(AproachSpeed, 1.0)), min=1.0, max=3.0), luego setea TimeSinceLastMove, WasInStopContextLastCheck, setea LastLocation y llama a CheckNavViolation(SafeDelta = 1.0, bCountSeconds = true).

Si A es true, pasa a un Branch (llamémosle B) cuya condition es NOT(ResumeGraceActive). Si B es False, setea ResumeGraceTime como ResumeGraceTime - 1.0. Luego, hay Branch cuya condition es Throttledelaicontroller > 0.0, si es True, setea Debug_Acum_ThrottleDuringGraceTime como Debug_Acum_ThrottleDuringGraceTime + Throttledelaicontroller, si es false, setea Debug_Acum_BrakeDuringGraceTime como Debug_Acum_BrakeDuringGraceTime + ABS(Throttledelaicontroller). Luego pasa a un Branch (llamémosle C), cuya condition es ResumeGraceTime <= 0.0. Si es false, sigue el flujo, si es true, setea ResumeGraceActive como False. Tras ambos caminos, setea TimeSinceLastMove, WasInStopContextLastCheck, setea LastLocation y llama a CheckNavViolation(SafeDelta = 1.0, bCountSeconds = true).

Si B es true, primero pasa a un Branch cuya condition es: NOT(IsMovingOnGround) AND (TimeSinceLastMove >= 1), si es false sigue el flujo, si es true, llama a KillCar cuya razón de muerte es FLIPPED.

Luego pasa a un Branch (llamemosle D), cuya condición es: ((DistanceVector(GetActorLocation, LastLocation)) < (selectfloat (a=(maxspeedx0.05), b=(maxspeedx0.05xfrontobstacledistance), picka=(frontobstacledistance>0.65))) AND (NOT(ForceStop del AiController)) AND (NOT((bIsTrafficLightActive del AiController) AND ((TrafficLightState del AiController)<=0.5))). 

Si el branch D es True, hace un ++ de TimeSinceLastMove, y luego llama a ApplyLazyMovePenalty. 

Por último hay un Branch (E) cuya condition es TimeSinceLastMove>=3, si es false setea WasInStopContextLastCheck, setea LastLocation y llama a CheckNavViolation(SafeDelta = 1.0, bCountSeconds = true), si es true, setea Debug_DeathPenalty como LocalLazyPenaltydeloutput y llama a killcar cuya cause es "LAZY". 

Si el branch D es false, setea TimeSinceLastMove, WasInStopContextLastCheck, setea Last Location y llama a CheckNavViolation(SafeDelta = 1.0, bCountSeconds = true).




EventActorBeginOverlap:

Lo primero hay un isvalid? de otheractor. Luego hay un CasttoBPIntersectionBorders con el otheractor. Si el cast es válido, llama a CheckMatch del cast, luego pasa a un branch (llamemosle A), cuya condition es Matchdeloutput, si es true, primero hay un Branch cuya condition es bIsRoundabout del cast, si es false sigue el flujo, si es true, hace un ++ de Debug_CollisionsInRoundabout. Luego, se llama a ApplyCollisionDeathPenalty con CollisionCause = ("COLLISION_WITH_" + getDisplayName de otherActor).

Si A es false, pasa a un branch (llamemosle B) cuya condition es ActorHasTag=Semaforo del otheractor. Si es true, hace un casttoBPTrafficLight y del cast setea ActualTrafficLight.

Si B es false, hace un CasttoBP_VehicleAI2(OtherActor), si el cast es válido, pasa a un Branch cuya condition es IsDead(cast) OR IsDead(Self). Si es true vacío, si es false, llama a ApplyCarOverlapPenalty(OtherCar=cast).

Si el cast de intersectionBorders es false, pasa a un branch cuya condition es (ActorHasTag=Obstaculo) OR (ActorHasTag=Discontinua). Si es true llama a ApplyCollisionDeathPenalty con CollisionCause = ("COLLISION_WITH_" + getDisplayName de otherActor).

si es false, primero pasa a un Branch cuya condition es ActorHasTag=FueraDeCalle, si es true, hace un ++ de NavOverlapCount, si es false, pasa al branch B.




EventActorEndOverlap:

Lo primero hay un CastToTrafficLight(OtherActor). Si el Cast es fallido, pasa al Branch de FueraDeCalle. Si el Cast es válido, pasa a un Branch cuya condition es casttotrafficlight == ActualTrafficLight. Si es true setea ActualTrafficLight como vacío y setea ForceStopdelAiController como False. 

Ambos caminos del Branch, dan al Branch de FueraDeCalle, cuya condition es ActorHasTag("FueraDeCalle", OtherActor). Si el Branch es false, vacio, si es true setea NavOverlapCount como MAX(0, NavOverlapCount - 1).

Por último hay otro Branch cuya condition es NavOverlapCount == 0. Si es false vacío, si es true setea AccumNavPenalty como 0.0.

ApplyAlignmentPenalty

inputs:

SafeDelta float


variables locales:

DeltaAlignmentPenalty float

MisalignmentFactor float

ProximityFactor float


flujo:

primero se setea MisalignmentFactor como 1.0 - MAX(0.0, DotProduct(GetActorForwardVector(Self), StopLineForwardVector(AiController))).

Luego, se setea ProximityFactor como 1.0 - ClampFloat(value = Distance(GetActorLocation(Self), StopTargetLocation(AiController)) / MAX(SensorEntryDistance(AiController), 1.0), min = 0.0, max = 1.0).

Luego, se setea DeltaAlignmentPenalty como MisalignmentFactor × ClampFloat(value = ABS(GetForwardSpeed(Self)) / MaxSpeed(AiController), min=0.0, max=1.0) × MaxSpeed(AiController) × ClampFloat(value = AproachSpeed(Self) / MaxSpeed(AiController), min=0.0, max=1.0) × SafeDelta × ProximityFactor.

Luego, setea Fitness como Fitness - DeltaAlignmentPenalty.

Y por último, setea Debug_Acum_AlignmentPenalty como Debug_Acum_AlignmentPenalty + DeltaAlignmentPenalty.

CheckNavViolation

inputs:

SafeDelta float

bCountSeconds boolean


variables locales:

DeltaNavPenalty float



flujo:

lo primero hay un Branch cuya condition es NavOverlapCount > 0:

si es false, setea NavOutsideSeconds como 0 y AccumNavPenalty como 0.0.

si es true, pasa a otro Branch cuya condition es bCountSeconds.

si bCountSeconds es true, hace un ++ de NavOutsideSeconds, luego pasa a otro Branch cuya condtion es NavOutsideSeconds >= Truncate(NavPenaltyDeathThreshold / MaxVisionDistancedelAicontroller). 

si es true setea Debug_DeathPenalty como AccumNavPenalty y llama a KillCar(Cause = "NAV_VIOLATION"). 

si este branch es false, hace un ++ de NavTotalOutsideSeconds, luego pasa a otro Branch cuya condtion es NavTotalOutsideSeconds >= Truncate(NavPenaltyDeathThreshold / MaxSpeeddelAicontroller), que si es false vacío y si es true setea Debug_DeathPenalty como AccumNavPenalty y llama a KillCar(Cause = "NAV_VIOLATION").


si bCountSeconds es false, se setea DeltaNavPenalty como MAX(Clamp((ForwardSpeed/MaxSpeed), 0, 1), SafeDivide(MaxVisionDistance, NavPenaltyDeathThreshold))^2 × MaxVisionDistance × SafeDelta. 

Luego se setea Fitness como Fitness - DeltaNavPenalty. Luego se setea AccumNavPenalty como AccumNavPenalty + DeltaNavPenalty. 

Luego se setea Debug_Acum_NavPenalty como Debug_Acum_NavPenalty + DeltaNavPenalty. 

Luego pasa a un Branch cuya condition es AccumNavPenalty >= NavPenaltyDeathThreshold, si es false sigue el flujo, si es true setea Debug_DeathPenalty como AccumNavPenalty y llama a KillCar, esta razón de muerte es NAV_VIOLATION.
