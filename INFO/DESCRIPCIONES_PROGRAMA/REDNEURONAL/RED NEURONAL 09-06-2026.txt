PrepareNetworksInputs:


variables locales:

outputs:

ReadyInputs array de floats



flujo:

lo primero hay un cleararray de InputsforFrame. Luego, ace un add array InputsforFrame con el valor de la velocidad. Este valor se saca con el getforwardspeed de vehiclemovementcomponent dividido entre maxspeed (ahora mismo 1000). el resultado de esta división se mete en un nodo Clamp (float) de 0.0 a 1.0 que es el valor que añadimos al array. Lo siguiente en el flujo es un add array InputsforFrame con el valor de la velocidad angular. Este valor se saca del nodo getphysicsangularvelocityindegrees del mesh del vehiculo controlado, y devuelve solo el valor z. este valor z se mete en un map range clamped, de -90 a 90 y devuelve un valor de -1 a 1, que es el valor de la velocidad angular que añadimos al InputsforFrame. 

después, hay un GetSensorsData cuyo output es SensorsReadings que setea la variable local SensorsArray. tras esto va un for each loop que recorre SensorsArray. En el loop body se añade a InputsforFrame el arrayelement del loop. 
Al completarse dicho loop, se setea la variable FrontObstacle distance como un get del SensorsArray cuyo índice es SensorCount/2. Después, se setea DistSensorIzda como un get del SensorsArray cuyo índice es 0. Después se setea DistSensorDcha como un get del SensorsArray cuyo índice es SensorCount-1. 

Luego, hay un addInputsforFrame, al que se le añade TrafficLightState. Luego, hay un addInputsforFrame, al que se le añade SelectFloat(A=ClampFloat(value=(DistanceVector(getActorLocation(MyControlledCar), StopTargetLocation)/MAX(SensorEntryDistance, 1.0)), 0.0, 1.0), B=1.0, PickA=(IsTrafficLightActive OR ForceStop)). 

Luego, hace un addInputsforFrame, al que se le añade un SelectFloat(A=1.0, B=0.0, pickA=(TrafficControlType==0)). Luego, hace un addInputsforFrame, al que se le añade un SelectFloat(A=1.0, B=0.0, pickA=(TrafficControlType==1)). Después, hay un add InputsforFrame al que se le añade un valor de 1.0 (bias). Por ultimo en el flujo está el ReturnNode, que devuelve ReadyInputs como InputsforFrame.

ProcessAI:


Inputs:

AllInputs array de floats



VariablesLocales:

HiddenLayerValues array de floats

TempSum float

NumHiddenLayers Integer

FinalOutputs array de floats

Inputs array de floats

WItH array de floats

WHtO array de floats



ouputs:

Steering float

Throttle float




flujo:

setea numhiddenlayerslocal como NumHidden. Luego hay un Branch cuya condition es ((numhiddenlayerslocal <= 0) OR (numhiddenlayerslocal > 512)). Si es false, sigue el flujo, si es true, pasa a la dinámica de "GuardExit": 
hace un ++ de GuardFireCount, luego pasa a un Branch cuya condition es GuardFireCount>= 5. Si es true llama a KillCardemycontrolledCar y cause="INVALID_BRAIN", y luego hace un ReturnNode(Steering = 0.0, Throttle = 0.0). Si fuera false, hace un ReturnNode(Steering = getSteering, Throttle = getThrottle).  

Si el guard no ha saltado, setea Inputs como AllInputs. Después hay un Branch cuya condition es Length(Inputs) != Truncate(SafeDivide(A=Length(WeightsIH), B=MAX(NumHidden, 1))). Si es false sigue el flujo, si es true pasa a "GuardExit".  

Si es seguro continuar, luego, hace cleararrays de HiddenLayerValues y luego de FinalOutputs.

Después, setea WItH como WeightsIH y setea WHtO como WeightsHO. Luego, hay un Branch cuya condition es Length(WItH) != (numhiddenlayerslocal × Length(Inputs)). Si es false sigue el flujo, si es true, pasa a "GuardExit".

Después si es seguro, pasa a otro Branch cuya condition es Length(WHtO) != (numhiddenlayerslocal × 2). Si es false sigue el flujo, si es true, pasa a "GuardExit".

Después de comprobar que no ha saltado ningún guard, pasa a un Branch, cuya condición es length(WeightsIH) <= 0. Si es false sigue el flujo, si es true, pasa a "GuardExit".

tras el false del Branch hay un forloop, cuyo índice va desde 0 hasta numhiddenlayers-1. El loop body es el siguiente: primero se setea TempSum a 0.0, luego, hay otro ForLoop cuyo índice va desde 0 hasta (length(Inputs) - 1). En el loopbody se setea TempSum como TempSum + (GetWItH((Length(Inputs) x indexloopexterno) + indexloopinterno) x GetInputs(indexloopinterno)).

Tras completar el loop interno, se hace un addarray de HiddenLayerValues cuyo item añadido es tanh(TempSum).

Cuando se completa el bucle externo (el que iba desde 0 hasta numhiddenlayers-1), primero hay un Branch, cuya condition es Length(HiddenLayerValues) != NumHiddenLayerslocal, Si es false sigue el flujo, si es true, pasa a "GuardExit".

Luego, hay otro forloop, cuyo indice va desde 0 hasta 1. El loopbody de este bucle es el siguiente: primero se setea TempSum 0.0, luego, hay otro forloop cuyo indice va desde 0 hasta (length(HiddenLayerValues) - 1). En el loopbody de este bucle interno se setea TempSum como TempSum + (GetWHtO(indexloopinterno + (NumHiddenLayers x indexloopexterno)) x GetHiddenLayerValues(indexloopinterno)). 

Tras completar este bucle interno, se hace un addarray de FinalOutputs cuyo item añadido es un tanh(TempSum). 

Tras completar el bucle externo (el que iba de 0 hasta 1), primero setea GuardFireCount como 0, y luego está un ReturnNode, donde se devuelve Steering como GetFinalOutputs(0) y Throttle como GetFinalOutputs(1).
