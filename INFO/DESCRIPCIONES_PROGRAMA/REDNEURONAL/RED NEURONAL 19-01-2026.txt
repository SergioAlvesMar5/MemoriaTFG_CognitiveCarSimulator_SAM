GetSensorsData:

esta función no es necesaria tocarla, pero simplemente quiero informarte de que hace.
Esta función devuelve un array de floats SensorReadings. este array se genera a partir de un for loop desde 0 hasta sensor count menos 1 y aplicado truncate. Este loop body realiza un sphere trace by channel, y hace un add array a SensorReadings, con la distancia del sensor. esta distancia, varia si ha impactado o no con algún obstáculo. si ha tocado con algo, es la distancia al obstáculo entre la visionmaxima (ahora mismo 3000.0). si no ha impactado en nada, el valor es 1.0

PrepareNetworksInputs:

flujo:

limpia el array de floats InputsforFrame. Hace un add array InputsforFrame con el valor de la velocidad. Este valor se saca con el getforwardspeed de vehiclemovementcomponent dividido entre maxspeed (ahora mismo 1500). el resultado de esta división se mete en un nodo Clamp (float) de 0.0 a 1.0 que es el valor que añadimos al array. Lo siguiente en el flujo es un add array InputsforFrame con el valor de la velocidad angular. Este valor se saca del nodo getphysicsangularvelocityindegrees del mesh del vehiculo controlado, y devuelve solo el valor z. este valor z se mete en un map range clamped, de -90 a 90 y devuelve un valor de -1 a 1, que es el valor de la velocidad angular que añadimos al InputsforFrame. tras esto va un for loop, cuyo index va desde 0 hasta SensorCount-1. En el loop body se llama a GetSensorsData que devuelve el array con SensorReading, tras eso hay un nodo add InputsforFrame al que se le añade el valor del sensor. este valor se saca de un get del array cuyo índice es el index del loop. Al completarse dicho loop, se setea la variable FrontObstacle distance como un get del SensorReadings cuyo índice es SensorCount/2. Después, se setea DistSensorIzda como un get del SensorReadings cuyo índice es 0. Después se setea DistSensorDcha como un get del SensorReadings cuyo índice es SensorCount-1. Por ultimo en el flujo está el ReturnNode, que tiene como output ReadyInputs al que se le introduce InputsforFrame.

ProcessAI:

Inputs:

AllInputs array de floats


VariablesLocales:

HiddenLayerValues array de floats

TempSum float

NumHiddenLayers Integer

FinalOutputs array de floats

Inputs array de floats

WItH array de floats

WHtO array de floats


flujo:

setea numhiddenlayerslocal con la variable global NumHidden (12). después setea Inputs con el array de entrada de la función AllInputs. Después, setea WItH ccon WeightsIH y setea WHtO con WeightsHO. Después hay un Branch, cuya condición es que WeightsIH tenga una length mayor que 0. si el Branch es true continua si es false está vacio.
tras el true del Branch hay un for loop, cuyo índice va desde 0 hasta numhiddenlayers-1. El loop body es el siguiente, primero se setea TempSum a 0.0, luego, hay otro For Loop cuyo índice va desde 0 hasta Inputs length menos 1. En el loop body se setea TempSum. El valor de este temp sum se saca a partir del siguiente calculo, se coge el length de Inputs y se multiplica por el Index del primer loop (exterior), el resultado de esta multiplicación se suma con el index del bucle interno o actual. el resultado de esta suma, se utiliza como indice para un get del array WItH. La salida de este get, se multiplica por un get del array Inputs cuyo indice es el index del loop interno o actual. el resultado de esta multiplicacin de los dos gets se suma a TempSum. El resultado de esta suma es el que se utiliza para setear TempSum. Tras completar el loop interno, se hace un add array HiddenLayerValues cuyo item añadido es un Clamp (Float) entre -1 y 1 cuyo value es TempSum.
Cuando se completa el bucle externo (el que iba desde 0 hasta numhiddenlayers-1) hay otro for loop, cuyo indice va desde 0 hasta 1. El loop body de este bucle es el siguiente, primero se setea TempSum 0.0, luego, hay un loop for cuyo indice va desde 0 hasta HiddenLayerValues length menos 1. En el loop body de este bucle interno se setea TempSum. El valor que setea TempSum es el siguiente, Se multiplica el index del for loop interno x2, el valor de esta multiplicación se suma a el index del bucle externo. El resultado de esta suma se usa de indice para un nodo Get del array WHtO. El resultado de este get se multiplica con el resultado de otro nodo get del array HiddenLayerValues cuyo indice es el index del bucle interno. el resultado de la multiplicacion de los get, se suma a TempSum. El resultado de esta suma es el valor que setea TempSum.
Tras completar este bucle interno, se hace un add array FinalOutputs cuyo item añadido es un Clamp (Float) entre -1 y 1 cuyo value es TempSum. Tras completar el bucle externo (el que iba de 0 hasta 1) está el ReturnNode, que tiene como salidas Steering y Throttle. Steering se saca de un get FinalOutputs cuyo indice es 0 y Throttle se saca de un get FinalOutputs cuyo indice es 1.

Estas salidas son las que fuera en el event graph del controller, directamente van al vehiculo, steering se pasa a un nodo Set Steering Input, y Throttle primero se comprueba si es mayor o igual a 0.0. Si es positivo se pasa el valor abs a Set Throttle Input y se setea Set Brake Input a 0.0 y si es negativo se pasa el valor absoluto a Set Brake Input y se setea Set Throttle Input a 0.0
