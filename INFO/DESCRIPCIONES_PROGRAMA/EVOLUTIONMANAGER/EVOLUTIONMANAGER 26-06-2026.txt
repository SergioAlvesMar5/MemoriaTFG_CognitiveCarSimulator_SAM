cleararray de spawnpointslist y de spawnbag. luego hay un GetAllActorsofClassBP_SpawnPoints, cuyo loop body es: isvalid del arrayelement. si es valido, pasa a un Branch cuya condition es IsTrainingMode.
Si es true, hay un branch cuya condición es ValidForTrain del arrayelement. Si es true hay un addarrayuniquespawnpointslist al que se añade el arrayelement. 
Si es false, hay un branch cuya condición es ValidForTest del arrayelement. Si es true hay un addarrayuniquespawnpointslist al que se añade el arrayelement.

Al completarse el loop, se setea PopulationSize como lenghtspawnpointslist, luego hay un Branch cuya condición es PopulationSize>0, si es true, hay un forloop desde 0 hasta populationsize-1, cuyo loop body es: un addarray de spawnbag, al que se le mete un ((indexdelloop)%(lenght de spawnpointslist)). Al completarse el loop, hay un shufflearray de spawnbag.

WeightsDifferentFromLoaded


inputs:

CandidateIH array float

CandidateHO array float


outputs:

Different boolean


variables locales:

Index integer

Epsilon float = 0.000001




flujo:


lo primero hay un Branch cuya condition es (Length(CandidateIH) != Length(Loaded_W_IH)) OR (Length(CandidateHO) != Length(Loaded_W_HO)). Si es true hay un ReturnNode(Different = True).

si es false, pasa a un ForLoop(FirstIndex = 0, LastIndex = (Length(CandidateIH) - 1)), cuyo loopbody es: setea Index como indexdelloop, luego hay un Branch cuya condition es ABS(CandidateIH[Index] - Loaded_W_IH[Index]) > Epsilon, si es false vacío, si es true hay un ReturnNode(Different = True).

al completarse el loop, pasa a otro ForLoop(FirstIndex = 0, LastIndex = (Length(CandidateHO) - 1)), cuyo loopbody es: setea Index como indexdelloop, luego hay un Branch cuya condition es ABS(CandidateHO[Index] - Loaded_W_HO[Index]) > Epsilon, si es false vacío, si es true hay un ReturnNode(Different = True).

al completarse este último loop, hay un ReturnNode(Different = False).
