EventGraph:





BeginPlay:

lo primero hay un Parent:BeginPlay, luego hay un CasttoBP_aicontroller de GetController. Del cast lo primero hay un isvalid?, si es así se setea AiController con el cast. Luego hay un setColor. Luego hay un Casttochaoswheeledvehiclemovementcomponent de vehiclemovementcomponent. Del cast se setea AsChaosWheeledVehicleMovementComponent. 

Luego se setea LastLocation con GetActorLocation. Después de eso hay un SetTimerbyEvent, que tiene como opciones Time 1.0, y looping en True.




EventTick:

Lo primero """hay un delay cuya duration=Deltasecondsdeldelay. Luego""" hay un Parent:Tick de 0.0. Luego hay un SetAngularDamping cuyo target=mesh, InDamping=(SelectFloat(a=0.0, b=3.0, picka=Ismovingonground(AsChaosWheeledVehicleMovementComponent))). Luego hay un Branch cuya condition es NOT(IsDead), si es false, vacío, si es true sigue el flujo. Luego hay un Branch cuya condition es (NOT(TrainingMode de AiController)) AND (TimeAlived>MaxTimeAlived). Si es true llama a KillCar cuya razón de muerte es TIMEFINISHED, si es false, pasa a otro branch cuya condition es TimeAlived>180. Si es true, llama a KillCar cuya razón de muerte es TIMEFINISHED, si es false, llama a la función UpdateFitness.




CheckStuck:

Está conectada con el hilo rojo que sale del SetTimerbyEvent. 

El flujo de este evento es: Branch cuya condition es IsDead, si es true vacío, si es false, time alived ++. Luego un Branch cuya condición es si time alived es mayor que 3. Si el Branch es false, setea WasInStopContextLastCheck como (ForceStop del AiController) OR ((bIsTrafficLightActive del AiController)AND((TrafficLightState del AiController)<=0.5)), setea TimeSinceLastMove como 0 y setea Last Location como getactorlocation. 

Si es true, pasa a un Branch (llamémosle A) cuya condition es NOT(NOT((ForceStop del AiController) OR ((bIsTrafficLightActive del AiController)AND((TrafficLightState del AiController)<=0.5))) AND WasInStopContextLastCheck). 

Si A es false, setea TimeSinceLastMove, WasInStopContextLastCheck y setea Last Location.

Si A es true, pasa a un Branch (llamemosle B), cuya condición es: ((DistanceVector(GetActorLocation, LastLocation)) < (selectfloat (a=(maxspeedx0.05), b=(maxspeedx0.05xfrontobstacledistance), picka=(frontobstacledistance>0.65))) AND (NOT(ForceStop del AiController)) AND (NOT((bIsTrafficLightActive del AiController) AND ((TrafficLightState del AiController)<=0.5))). 

Si el branch B es True, setea Fitness como Fitness - MAX(0.0, (selectfloat (a=(maxspeedx0.05), b=(maxspeedx0.05xfrontobstacledistance), picka=(frontobstacledistance>0.65)) - DistanceVector(GetActorLocation, LastLocation))) """((1- (1- (ABS(distsensordcha - distanciasensorizda)))) x maxspeed)""", setea Debug_Acum_LazyPenalty como Debug_Acum_LazyPenalty + MAX(0.0, (selectfloat (a=(maxspeedx0.05), b=(maxspeedx0.05xfrontobstacledistance), picka=(frontobstacledistance>0.65)) - DistanceVector(GetActorLocation, LastLocation))) """((1- (1- (ABS(distsensordcha - distanciasensorizda)))) x maxspeed)""" y hace un ++ de TimeSinceLastMove. Por último hay un Branch cuya condition es TimeSinceLastMove>=3, si es false setea WasInStopContextLastCheck y setea Last Location, si es true, setea Debug_DeathPenalty como MAX(0.0, (selectfloat (a=(maxspeedx0.05), b=(maxspeedx0.05xfrontobstacledistance), picka=(frontobstacledistance>0.65)) - DistanceVector(GetActorLocation, LastLocation)))G """((1- (1- (ABS(distsensordcha - distanciasensorizda)))) x maxspeed)""" llama a killcar cuya cause es "LAZY". 

Si el branch B es false, setea TimeSinceLastMove, WasInStopContextLastCheck y setea Last Location.




EventActorBeginOverlap:

Lo primero hay un isvalid? de otheractor. Luego hay un CasttoBPIntersectionBorders con el otheractor. Si el cast es válido, pasa a un branch (llamemosle A), cuya condition es Trigger del cast == CurrentTrigger del AiController. Si A es true pasa a otro branch (llamemosle B) cuya condition es DirectionIntent del cast == IntersectionDecision de AiController. Si B es True, se setea Fitness como fitness - ((getforwardspeed^2/(maxspeed del aicontroller))x(1-(frontobstacledistnace del aicontroller))). Luego se setea DebugDeathPenalty como ((getforwardspeed^2/(maxspeed del aicontroller))x(1-(frontobstacledistnace del aicontroller))). Luego se llama a KillCar cuya Cause es "COLLISION_WITH_" + getDisplayName de otherActor.

Si A o B son false, pasan a un branch (llamemosle C) cuya condition es ActorHasTag=Semaforo del otheractor. Si es true, hace un casttoBPTrafficLight y del cast setea ActualTrafficLight.

Si el cast de intersectionBorders es false, pasa a un branch cuya condition es (ActorHasTag=Obstaculo) OR (ActorHasTag=Discontinua). Si es true pasa a setear el fitness y matar al coche, si es false, pasan al branch C.




EventActorEndOverlap:

Lo primero hay un CastToTrafficLight, luego hay un branch cuya condition es si el cast == ActualTrafficLight. Si false, vacío, si es true SeteaActualTrafficLight como vacío y setea ForceStop del AiController como False.

UpdateFitness: 



flujo:

Lo primero hay un Branch cuya condición es si getforwardspeed es mayor o igual que maxspeed(1000)*frontobstacledistance(1 lejos, 0 cerca), si es false sigue el flujo, si es true setea el fitness como fitness - ((Clamp(value=(forwardspeed/maxspeed),min=0.0,max=1.0))^2)*worlddeltaseconds*maxspeed*(1-frontobstacledistnace)), setea DebugAcumPenaltyVelocidad como DebugAcumPenaltyVelocidad + ((Clamp(value=(forwardspeed/maxspeed),min=0.0,max=1.0))^2)*worlddeltaseconds*maxspeed*(1-frontobstacledistnace)), y sigue el flujo.


Luego, hay un Branch cuya condition es forwardspeed < 0.0, si es false, setea ReverseTime como 0.0 y sigue el flujo. Si es true, setea ReverseTime como ReverseTime + WorldDeltaSeconds, luego setea Fitness como Fitness - ((Clamp(value=(ABS(forwardspeed)/maxspeed),min=0.0,max=1.0))^2)*worlddeltaseconds*maxspeed), y setea Debug_Acum_ReversePenalty como Debug_Acum_ReversePenalty + ((Clamp(value=(ABS(forwardspeed)/maxspeed),min=0.0,max=1.0))^2)*worlddeltaseconds*maxspeed).


tras ambos caminos, luego está el Branch cuya condición es si (frontobstacledistance>=0.99 AND steering*horizontaldistdiff(que es distsensordcha-distsensorizda)<0.0 AND ABS(horizontaldistdiff) > 0.075) OR ((forwardspeed < -5.0) AND (SafeDivide(A=(ABS(ForwardSpeed)*ReverseTime), B=MAX(FrontObstacleDistance*MaxVisionDistance, 1.0)) > Clamp(SafeDivide(A=ABS(ForwardSpeed), B=MAX(AproachSpeed, 1.0)), 0.0, 1.0))). 

si es true, hay otro Branch cuya condición es si (frontobstacledistance>=0.99 AND steering*horizontaldistdiff(que es distsensordcha-distsensorizda)<0.0 AND ABS(horizontaldistdiff) > 0.075) AND ((forwardspeed < -5.0) AND (SafeDivide(A=(ABS(ForwardSpeed)*ReverseTime), B=MAX(FrontObstacleDistance*MaxVisionDistance, 1.0)) > Clamp(SafeDivide(A=ABS(ForwardSpeed), B=MAX(AproachSpeed, 1.0)), 0.0, 1.0))). 

si es true, setea fitness como fitness - (ABS(forwardspeed)+(maxvisiondistance(3000.0)*ABS(steering)*ABS(horizontaldistancediff))), setea DebugDeathPenalty como (ABS(forwardspeed)+(maxvisiondistance(3000.0)*ABS(steering)*ABS(horizontaldistancediff))) y llama a KillCar, esta razón de muerte es REVERSING_WRONG.

Si el Branch es false, pasa a otro Branch cuya condición es (forwardspeed < -5.0) AND (SafeDivide(A=(ABS(ForwardSpeed)*ReverseTime), B=MAX(FrontObstacleDistance*MaxVisionDistance, 1.0)) > Clamp(SafeDivide(A=ABS(ForwardSpeed), B=MAX(AproachSpeed, 1.0)), 0.0, 1.0)). Si es true, setea fitness como fitness - ABS(forwardspeed), setea DebugDeathPenalty como ABS(forwardspeed) y llama a killcar, esta razón de muerte es REVERSE. 

Si el Branch es true, setea wrongcorrectiontimer como getworlddeltaseconds + wrongcorrectiontimer. Luego setea fitness como fitness - (maxvisiondistance(3000.0)*ABS(steering)*ABS(horizontaldistancediff)*worlddeltaseconds). Luego setea DebugAcumPenaltyCorrectingWrong como DebugAcumPenaltyCorrectingWrong + (maxvisiondistance(3000.0)*ABS(steering)*ABS(horizontaldistancediff)*worlddeltaseconds) y ++ de DebugTicksTotal. 


luego hay un Branch cuya condition es (wrongcorrectiontimer>0.1)AND(TrainingMode del aicontroller). Si es true, setea DebugDeathPenalty como 0.0, y llama a killcar, esta razón de muerte es CORRECTINGWRONG. Si el Branch es false, setea PrevHorizontalDistDiff como ABS(HorizontalDistDiff) y termina la función.




En caso de que el Branch de (frontobstacledistance>=0.99 AND steering*horizontaldistdiff(que es distsensordcha-distsensorizda)<0.0 AND ABS(horizontaldistdiff) > 0.075) OR ((forwardspeed < -5.0) AND (SafeDivide(A=(ABS(ForwardSpeed)*ReverseTime), B=MAX(FrontObstacleDistance*MaxVisionDistance, 1.0)) > Clamp(SafeDivide(A=ABS(ForwardSpeed), B=MAX(AproachSpeed, 1.0)), 0.0, 1.0))) salga false, se pasa  a otro Branch dentro ya del cálculo normal del fitness:


hay un Branch (A) cuya condition es ForceStopdeaicontroller OR (IsTrafficLightActivedelaicontroller AND (TrafficLightStatedelaicontroller<=0.5)), si es false sigue el flujo (a setear wrongcorrectiontimer y demás), si es true, pasa a otro Branch (B).

Branch (B) cuya condition es ((bIsTrafficLightActive del aicontroller) AND ((TrafficLightState del aicontroller)<=0.5), si es false sigue el flujo, si es true, setea Fitness como Fitness - (((Clamp(value=(forwardspeed/maxspeed),min=0.0,max=1.0))^2)*worlddeltaseconds*maxspeed*SelectFloat(a=0.3, b=1.0, picka=(TrafficLightState==0.5) y setea Debug_Acum_PenaltyTrafficViolation como Debug_Acum_PenaltyTrafficViolation + (((Clamp(value=(forwardspeed/maxspeed),min=0.0,max=1.0))^2)*worlddeltaseconds*maxspeed).

tras ambos caminos del branch B, hay un branch (C), cuya condition es TrafficControlTypedelaicontroller<=1. si es false sigue el flujo (a setear wrongcorrectiontimer y demás).

Si es true, hay un set Fitness como Fitness - DeltaSeconds x AproachSpeed x (1- ClampFloat(value=SafeDivide(a= Distance(GetActorLocation, StopTargetLocationdelAiController), b=MAX(Sensorentrydistancedeaicontroller, 1.0)), min=0.0, max=1.0)) x Clamp(value=SafeDivide(a=ABS(GetForwardSpeed), b=MAX(AproachSpeed, 1.0)), min=0.0, max=1.0). tras eso se setea Debug_Acum_CreepingPenalty como Debug_Acum_CreepingPenalty + DeltaSeconds x AproachSpeed x (1- ClampFloat(value=SafeDivide(a= Distance(GetActorLocation, StopTargetLocationdelAiController), b=MAX(Sensorentrydistancedeaicontroller, 1.0)), min=0.0, max=1.0)) x Clamp(value=SafeDivide(a=ABS(GetForwardSpeed), b=MAX(AproachSpeed, 1.0)), min=0.0, max=1.0).

Luego se setea WrongCorrectionTimer a 0.0. Luego hay otro Branch cuya condition es (ForceStopdeaicontroller OR (IsTrafficLightActivedelaicontroller AND TrafficLightStatedelaicontroller<=0.5)) AND (ABS(getfordwardspeed)<(Maxspeeddelaicontroller*0.005)), si es false sigue el flujo, si es true setea fitness como Fitness + (((LERP(A=1.0, B=(1-(ABS(distsensordcha-distsensorizda))), alpha=frontobstacledistance) x Getworlddeltaseconds x AproachSpeed x frontobstacledistance)+(((Clamp(value=(aproachspeed/maxspeed),min=0.0,max=1.0)^2)*maxspeed*frontobsacledistance^2*worlddeltaseconds))) x (1-ClampFloat(value=SafeDivide(Distance(GetActorLocation, StopTargetLocationdelaicontroller), MAX(SensorEntryDistancedelacontroller, 1.0)), min=0.0, max=1.0)).
Luego setea Debug_Acum_WaitBonus como Debug_Acum_WaitBonus + (((LERP(A=1.0, B=(1-(ABS(distsensordcha-distsensorizda))), alpha=frontobstacledistance) x Getworlddeltaseconds x AproachSpeed x frontobstacledistance)+(((Clamp(value=(aproachspeed/maxspeed),min=0.0,max=1.0)^2)*maxspeed*frontobsacledistance^2*worlddeltaseconds))) x (1-ClampFloat(value=SafeDivide(Distance(GetActorLocation, StopTargetLocationdelaicontroller), MAX(SensorEntryDistancedelacontroller, 1.0)), min=0.0, max=1.0)) y sigue el flujo. 




Ahora, se pasa al calculo del fitness:


LERP (A=1.0, B=(1 - (ABS(dist sensor dcha - dist sensor izda))), alpha=frontobstacledistance) x Get world delta seconds x get forward speed x front obstacle distance. ESTE VALOR ES A y se guarda como Debug_Acum_A.


MAX(0.0,(PrevHorizontalDistanceDiference- ABS(dist sensor dcha - dist sensor izda))). ESTE VALOR ES B y se guarda como Debug_Acum_B.


(1.0/MAX((dist sensor dcha + dist sensor izda),0.05)). ESTE VALOR ES C y se guarda como Debug_Acum_C.


(1.0 / (maxsteeringangle->D2R->SIN))). ESTE VALOR ES D y se guarda como Debug_Acum_D.


B x C x D = E y se guarda como Debug_Acum_E.


((Clamp(value=(forwardspeed/maxspeed),min=0.0,max=1.0)^2) * maxspeed * frontobsacledistance^2 * worlddeltaseconds) * MAX(Clamp(value=(ForwardSpeed/Maxspeeddelaicontroller), min=0.0, max=1.0) , Clamp(value=Throttledelaicontroller, min=0.0, max=1.0)). ESTE VALOR ES F y se guarda como Debug_Acum_F.


(A + E + F). El resultado de esto, se le suma al fitness y setea el fitness, además se guarda como Acum_NetNormal. 

Después del setfitness, y de los seteos de debugacum, hay un ++ DebugTicksNormal y un ++DebugTicksTotal. Luego, hay un Branch cuya condition es Throttle>=0.0, si es true setea Debug_Acum_ThrottlePositive como Debug_Acum_ThrottlePositive + Throttle*DeltaSeconds. Si es false sigue el flujo, si es true setea Debug_Acum_CoastTime como Debug_Acum_CoastTime + DeltaSeconds.

Tras eso, hay un Branch cuya condition es (ForceStopdeaicontroller OR (IsTrafficLightActivedelaicontroller AND TrafficLightStatedelaicontroller<=0.5)), si es false sigue el flujo, si es true, hace un ++ de Debug_Ticks_StopContext y pasa a otro Branch, cuya condition es Throttle>=0.0. Si es true, setea Debug_Acum_StopContextThrottle como Debug_Acum_StopContextThrottle + Throttle*DeltaSeconds. Si es false, setea Debug_Acum_StopContextBrake como Debug_Acum_StopContextBrake + ABS(Throttle)*DeltaSeconds.
